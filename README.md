# File-Name-Changing-scripts
A collection of scripts for batch renaming files. 

--How to use--
These files({}Capatalizer_program.py and {}File_prefix_remover.py files) work in their current directory. For example, if placed in documents 
                                                                                                                                    |--images 
                                                                                                                                       |--Family Photos
Only files in Family Photos would be affected

To change the prefix that is removed (in the {}File_prefix_remover.py file), edit the Pattern variable.
